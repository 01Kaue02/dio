Agrupamento de Seletores

Podemos aplicar as mesma regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula(,).

Assím, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesma regras!
